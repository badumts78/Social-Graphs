Python 3 graphviz layout gives "IndexError: list index out of range"#TITLE_END#Trying to use the graphviz layout `pos=nx.graphviz_layout(G, prog='dot')` with Python 3 gives an `IndexError: list index out of range`.  ``` /usr/local/lib/python3.4/dist-packages/networkx/drawing/nx_pydot.py in pydot_layout(G, prog, root, **kwds)     294      295         if isinstance(node,list): --> 296             node=node[0]     297         pos=node.get_pos()[1:-1] # strip leading and trailing double quotes     298         if pos != None:  IndexError: list index out of range ```  This can be fixed for Python 3 by removing the `.encode('utf-8')` on line `#292`:  ```     pydot_node = pydot.Node(make_str(n)).get_name() #.encode('utf-8') ```  Of course this fix probably breaks Python 2 but hopefully gives some indication as to the cause. Tested against latest `master`. See also [this SO post](http://stackoverflow.com/questions/32183964/python-3-networkx-draw-graphviz-does-not-work).
