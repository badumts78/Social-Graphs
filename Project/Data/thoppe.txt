`from_numpy_matrix` incorrect for multigraphs#TITLE_END#Using `from_numpy_matrix` on a multigraph will not add the additional edges. A minimal working example:  ``` import numpy as np import networkx as nx  A1 = [[0,2],[1,0]] g1 = nx.from_numpy_matrix(np.matrix(A1),nx.MultiDiGraph())  print "from_numpy_matrix edges: ", list(g1.edges())  g2 = nx.MultiDiGraph() g2.add_edge(0,1) g2.add_edge(0,1) g2.add_edge(1,0)  print "manual edges:            ", list(g2.edges()) ```  It is expected that these two should be the same. The output using version 1.9 is:  ``` from_numpy_matrix edges:  [(0, 1), (1, 0)] manual edges:             [(0, 1), (0, 1), (1, 0)] ```
