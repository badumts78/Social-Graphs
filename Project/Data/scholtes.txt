`add_edge` followed by `remove_edge` does not yield same graph  #TITLE_END#For example  ``` from networkx import *  g1 = Graph() g1.add_path([1,2,3])  g2 = Graph(g1)  print(is_isomorphic(g1,g2)) # => True  # Should do nothing? g2.add_edge(3,4) g2.remove_edge(3,4)  print(is_isomorphic(g1,g2)) # => False (bug?)  print(g1.edges()) # => [(1, 2), (2, 3)] print(g2.edges()) # => [(1, 2), (2, 3)] ```  This is not just a symptom of the graph copy (`g2 = Graph(g1)`).  For example:  ``` from networkx import *  g1 = Graph() g1.add_path([1,2,3])  g2 = Graph() g2.add_path([10,20,30])  print(g2.edges()) # => [(10, 20), (20, 30)] print(is_isomorphic(g1,g2)) # => True  # Should do nothing? g2.add_edge(10,40) g2.remove_edge(10,40)  print(g2.edges())  # => [(10, 20), (20, 30)] print(is_isomorphic(g1,g2)) # => False (bug?) ```   Python 3.5.2, NetworkX 1.11
