Serialization of objects when using json#TITLE_END#When using objects as nodes, the `__repr__` of that node doesn't appear to be escaped when exporting as JSON.  ``` python     #!/usr/bin/env python3     import networkx as nx     from networkx.readwrite import json_graph      class Foo:         def __init__(self, x):             self.x = x         def __repr__(self):             return "Foo|" + str(self.x)       G = nx.DiGraph()     G.add_edge(Foo(1), Foo(2))      print(json_graph.node_link_data(G))      # Returns     # {'directed': True,      #  'multigraph': False,      #  'links': [{'target': 0, 'source': 1}],      #  'nodes': [{'id': Foo|2}, {'id': Foo|1}], 'graph': {}}     #     # Expected     # {'directed': True,      #  'multigraph': False,      #  'links': [{'target': 0, 'source': 1}],      #  'nodes': [{'id': 'Foo|2'}, {'id': 'Foo|1'}], 'graph': {}}  (example placed here as GitHub doesn't allow python files to be attached) ```
