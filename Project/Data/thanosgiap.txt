For more details check out step 1 and 4 of https://networkx.org/documentation/latest/developer/contribute.html --> ## PLL Algorithm Implemented the PLL reachability algorithm for directed graphs in the pll_reachability.py module.  The pll_preprocess function processes the graph using 2-hop indexing, while pll_has_path checks for the existence of paths using precomputed labels. The index is being calculated using forward and backward BFS traversals and pruning already visited nodes to make the size of the index smaller.   Built on the implementation as detailed in this paper: [https://arxiv.org/pdf/2311.03542](https://arxiv.org/pdf/2311.03542)  This is my first pull request here, so I would gladly appreciate any comments and improvements.
