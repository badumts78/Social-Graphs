ENH: Add function for Hyper-Wiener Index#TITLE_END#### Proposal  New function, `networkx.hyper_wiener_index(G)`, to calculate the Hyper-Wiener Index of a graph.  The Hyper-Wiener Index ($WW$) is a well-known topological index used in chemical graph theory. It's a natural extension of the existing `nx.wiener_index`. Adding this function would enhance the library's utility for structure-property studies.  ### Proposed Implementation  The index is defined by the formula: $$WW(G) = \frac{1}{2} \sum_{u, v \in V(G)} [d(u,v) + d(u,v)^2]$$ where $d(u,v)$ is the shortest path distance between vertices $u$ and $v$.  This can be implemented efficiently by iterating through the results of `nx.all_pairs_shortest_path_length(G)`.  For consistency with `nx.wiener_index`, the function should raise a `NetworkXError` if the input graph is not connected.
